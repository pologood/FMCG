找汽配退换货规则

找汽配承诺符合以下情况，自客户收到商品之日起15日内可以退换货，客户可在线提交申请办理退换货事宜。具体退换货标准如下：

退换类别 具体描述 退货运费 备注
国家法律所规定的功能性故障或商品质量问题 经由厂家或有相关资质的第三方检测确认，并出具检测报告属于商品质量问题。 由卖家承担 当地无检测条件的商品，用户可与找汽配官方平台的质量维权部门联系，我们将代替您寻找专业机构进行责任认定和索赔等相关事宜。（备注：在责任界定结果出台之前所产生的费用暂由买家预先支付。）
1、物流是买卖双方自主选择，商品如与物流造成的破损、丢失应第一时间与物流公司联系解决，再报找汽配平台，我们会同时协助客户解决与物流公司的纠纷。2、商品与网站描述不符的情况应及时上报找汽配平台，我们将协调卖家及时为买家解决，并对卖家进行相应的处罚。 物流损指在运输过程中造成的损坏、漏液、破碎，经售后人员核查情况属实。 由相关物流公司承担 找汽配审核期间可能需要快递人员证明或要求您提供实物照片等，以便售后人员快速做出判断并及时处理。
其他原因 由于买家个人原因导致的退换货，应由买家与卖家协商解决。在包装完整且不影响二次销售的情况下，还需卖方认同或者卖方自愿参与平台的某项退换货活动。 由买家承担 由买家承担商品往返运费。

特别说明，以下情况无法办理退换货

1.过保产品。

2."平台无交易记录"的产品不能进入平台的退换货流程。

3.电子电气类产品及油品等特殊产品，按汽配行业标准不予退换。
